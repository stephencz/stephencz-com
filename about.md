This is the personal website/blog of Stephen Czekalski.
I've had a website, in some form or another, for several years.
I didn't start creating content for it until mid 2020's.

## Design 

The main objective of this website's design, both technical and aesthetic, is simplicity.

### Aesthetic

There isn't much to say about this website's aesthetic design because there isn't much to it.
When it comes to blogs I prefer the under-designed to the over-designed.
The purpose of a blog is to convey information. 
The less clutter on the screen the better.

When designing this website I took most of my inspiration from the simplicity of Aaron Swart's blog [Raw Thought](http://www.aaronsw.com/weblog/).
A narrow column of small serif font looks good and is easily read.
I tried to modernize his design by using less yellow, centering the column of text, and adding a header.


### Technical 

The technical requirements I have for this website are it should be small and fast.

I've, more or less, managed to meet these requirements.
I've made an effort to avoid third-party frameworks and libraries.
As of writing this the only external resources I use is the external CSS file and images if the page contains any.

A big part of keeping this website fast is by using the static site generator [Jekyll]().
Static websites are served to your web browser in the same form they are stored.
They are opposed to dynamic websites which have some overhead associated with each request.
Jekyll helps retain some of the conveniences, such as layouts and the ability to include HTML snippets, that are traditionally associated with dynamic websites.

You can find this website's source code on [GitHub](https://github.com/stephencz/stephencz-com).
