## Code Blocks, Breakouts, and Tables

This website is built with the static website generator Jekyll.
It can do some pretty neat syntax highlighting for code blocks.
Check it out:

~~~c++
#include <iostream>

using namespace std;

// THIS IS A REALLY LONG COMMENT TO TEST WHAT HAPPENS IF THE CODE OVERFLOWS THE SIZE OF THE CODE BLOCK WHAT DO YOU THINK WILL HAPPEN WHO KNOWS MAYBE WE WILL FIND OUT SOON

int main(int argc, char** argv)
{

    cout << "Hello World!" << endl;

    return 0;
}

~~~

In the original version of this post the above code block looks kind of tight.
For situations such as these I invented **Breakouts**!
Breakouts are a convient and way to break out of the main responsive container using pure CSS3.

{% breakout %}
~~~c++
#include <iostream>

using namespace std;

// THIS IS A REALLY LONG COMMENT TO TEST WHAT HAPPENS IF THE CODE OVERFLOWS THE SIZE OF THE CODE BLOCK WHAT DO YOU THINK WILL HAPPEN WHO KNOWS MAYBE WE WILL FIND OUT SOON

int main(int argc, char** argv)
{

    cout << "Hello World!" << endl;

    return 0;
}

~~~
{% endbreakout %}

Oh wait! That is wayyyyyy to big.
That is because I used a breakout without any additional styling or classes! 
This is useful for banner images.

{% breakout banner %}
<img src="https://i.ytimg.com/vi/MPV2METPeJU/maxresdefault.jpg" alt="An image of a golden retriever puppy">
{% endbreakout %}

For more serious applications such as making a slightly bigger area for code blocks we have to use the fixed breakout block and pass in a size.

{% breakoutfixed 800px %}
~~~c++
#include <iostream>

using namespace std;

int add(int a, int b)
{
  return a + b;
}

int main(int argc, char** argv)
{

    cout << "Hello World!" << endl;
    cout << add(2, 2) << endl;

    return 0;
}

~~~
{% endbreakoutfixed %}

I'm quite proud of the fact that this feature is pure CSS3 and is so flexible.
